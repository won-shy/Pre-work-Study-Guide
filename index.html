Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Child elements are elements nested inside other elements.
  • Parent elements holds all the children elements.
  • HTML elements typically start and close with an opening and closing tab, with all the content contained between.
  • The paragraph element represents a block of text or paragraph and wraps text on webpages.
  • The image element contains information about images displayed on the webpage.

CSS

  • Inline CSS: developer wants to style an element directly in the HTML file.
  • Internal CSS style sheet: contains CSS rules for webpage in head section.
  • External CSS style sheet: allows developers to keep all CSS rules in a separate file.
  • Selector: defines element or attributes to which the rules, or declarations, will apply.
  • Declarations: consists of CSS property we want to apply and the value of said property.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: switch to main branch
  • git pull origin main: pull down latest main branch
